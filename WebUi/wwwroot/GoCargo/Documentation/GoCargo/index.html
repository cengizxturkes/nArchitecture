HTML structure - top

This template build using Bootstrap framework version 3. All the information within main content is nested within a div with id- name "content". If you need more information you can read Bootstrap documentation on this link

 

1. Sample html structure for content
<div id="content">
	<div class="container">
		<div class="row">
			<div class="col-md-4">
				this is your content			
			</div>
			<div class="col-md-8">
				this is your content
			</div>
		</div>
	</div>
</div>

 

2. You can create column from 12 predefined grid size, then insert your content inside.
<div class="row">
  	<div class="col-md-1">your content</div>
	<div class="col-md-2">your content</div>
	<div class="col-md-3">your content</div>
	<div class="col-md-4">your content</div>
	<div class="col-md-5">your content</div>

CSS - top

All css files within this template is loaded by single file main.css. To modify template design and typography, you can edit files below:

 

@import url("style.css"); main css for this template, if you want to adjust template design you can edit this file

@import url("rev-settings.css"); custom css file for Revolution Slider plugin, you can modify slider typography from this file

 

Here is another css files used on this template (It's required to not modify files below).

 

@import url("import-font.css"); used to load font from external source like Google font
@import url("../fonts/font-awesome/css/font-awesome.css"); used to load font awesome icons
@import url("../fonts/et-line-font/style.css"); css from font awesome used to load font awesome icons
@import url("../rs-plugin/css/settings.css"); css file for Revolution Slider plugin
@import url("owl.carousel.css"); css for jQuery plugin "Owl Carousel"
@import url("owl.theme.css"); css for jQuery plugin "Owl Carousel"
@import url("animate.css"); css for animating template elements
@import url("magnific-popup.css"); css for jQuery plugin "Magnific Popup"

 

For HTML extended version, there is two more css files below:

 

@import url("custom.css"); custom css file for current template
@import url("color/blue.css"); color settings file for current template

Setting Instafeed (to show Instagram photo)

  • Open file name custom-instafeed.js in folder js
  • The only thing you'll need to get going is a valid client id from Instagram's API. You can easily register for one on Instagram's website.
  • Replace userID and clientID
    userId - Unique id of a user to get. Use with get: 'user'.
    clientId - Required. Your API client id from Instagram.
  • For more documentation read here http://instafeedjs.com

Setting Tweecool (to show Twitter)

  • Open file name custom-tweecool.js in folder js
  • Replace username and limit
    username - Your username
    limit - Number of tweets to show
  • For more documentation read here http://www.tweecool.com
